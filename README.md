## 📌주요기능
* 랜덤 채팅
  * 국가를 지정하여 랜덤으로 일대일 채팅 매칭
  * 친구 맺기
  * 파일 업로드/다운로드
  * 언어 번역
* 관심사 채팅
  * 열 세 개의 관심사 카테고리 분류
  * 카테고리에 따른 단체 채팅
  * 파일 업로드/다운로드
  * 언어 번역
* 친구 채팅
  * 친구를 맺은 상대와의 일대일 메신저
  * 친구 끊기
  * 파일 업로드/다운로드
  * 언어 번역 
* 화상 채팅 & 음성채팅 
  * 친구를 맺은 상대와의 일대일 화상 및 음성 채팅 
  * 언어 번역
* 마이페이지
  * 개인 정보 수정 
  * 프로필 사진 및 상태메시지 지정
  * 친구 리스트
